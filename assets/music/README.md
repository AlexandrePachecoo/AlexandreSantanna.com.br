# Trilhas instrumentais

Coloque aqui de 1 a 3 arquivos MP3 (Creative Commons / livres de direitos) com nomes:

- `trilha-1.mp3`
- `trilha-2.mp3`
- `trilha-3.mp3`

Sugestões de fonte: [Pixabay Music](https://pixabay.com/music/), [Free Music Archive](https://freemusicarchive.org/), [Uppbeat](https://uppbeat.io/).

Cada trilha é escolhida aleatoriamente quando a pessoa marca **"Música fofa instrumental"** no formulário. Se nenhum arquivo existir, o vídeo é renderizado em silêncio.

Recomendado: trilhas de 30s a 90s, mood "calmo / emocional / piano / acústico".
